## Using the Keyboard

Our system use western [solfège](https://en.wikipedia.org/wiki/Solf%C3%A8ge) in order make learning music more aproachable with a moveable do. If that made no sense to you please have [Julie Andrews (Playing Maria vo Trapp)](https://www.youtube.com/watch?v=drnBMAEA3AM&list=RDdrnBMAEA3AM&start_radio=1) explain it to you. 

So a few concepts that Julie Andrews dose not explain. 'Classical' music and all it's derivation of music notation asign frequencies letters: A, B, C, D, E, F, and G. Each not (sort of) can be moddifed with a sharp (#) or flat (b). If we say a note is sharp, it's higher than the note name. If we say the note is flat, the note is lower than the note name. For convoluted math reasons, this gets us wit 12 distinct notes. We only use 8 of these notes, which no matter what the first note's frequency is, will be called: Do Re Mi Fa So La Ti Do. These are layed out on your key board as A S D F G H J K. 

Now, you have two options: 
1. Mash some keys and have some fun
2. Find some songs with solfege notation and follow along. 

## Using the Panel

With a moveable do, any of these 12 notes can be your first note. In our system, there are technically infinate numbers of first notes, as we have you assign the home tone, the technical name for the first note, with a number.
